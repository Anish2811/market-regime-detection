# 📊 Market Regime Detection System

Machine Learning project for detecting stock market regimes:
- Bull Market 📈
- Bear Market 📉
- High Volatility ⚡
- Sideways Market ➖

## 🚀 Techniques Used
- KMeans Clustering
- Hidden Markov Model (HMM)
- Technical Indicators (RSI, MACD, ATR, Bollinger Bands)

## 📈 Dataset
- NSE: ^NSEI

## ⚙️ How to Run

```bash
pip install -r requirements.txt
python main.py
